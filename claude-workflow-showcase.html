03
廣告痛點分析 × TA 報告
Notion 諮詢名單 → 報告 → 廣告貼文
AI 共創 內容轉化

從 Notion 諮詢資料庫直接抓取成交客戶的記錄，讓 Claude 分析這些人的共同痛點、背景、決策動機，生成結構化的 TA 分析報告——再把報告作為素材，產出廣告貼文。從數據到廣告，一條龍。

執行流程
🗃串接 Notion 諮詢 DB
→
🔍篩出成交者記錄
→
📝痛點歸因分析
→
📄TA 分析報告
→
📣廣告貼文產出
為什麼這樣做？ 廣告文案最難的不是寫作，是「知道受眾在痛什麼」。用真實諮詢數據回推，比靠感覺猜受眾準確得多。
04
社群貼文月排程表
IG · Threads · /social-plan 指令
內容規劃 AI 共創

輸入 /social-plan，由社群行銷 Agent 接手，根據品牌定位與當月目標，規劃完整的月排程——主題方向、格式配比、每篇文案角度，直接輸出可用的表格版本。

輸出內容
月曆格式排程表 教育型 / 娛樂型 / 轉換型配比 每篇主題 + 文案角度 IG Reels 腳本方向 話題標籤策略
社群行銷 Agent 的思維框架 先確認這篇內容「為誰服務」（吸引新受眾 vs. 維繫現有受眾）→ 以信任建立路徑思考內容序列，不只看單篇效果。

06
Speed Reader · 文字轉語音工具
Deepgram Aura TTS · Karaoke 同步 highlight
工具開發

描述需求，Claude Code 直接串接 Deepgram Aura TTS API 寫出工具。輸入文字，生成語音的同時取得逐字時間軸，播放時像 Karaoke 一樣逐字 highlight，已讀文字變灰，方便跟讀或校對。

技術流程
📝輸入文字
→
🔊Deepgram TTS 生成音訊
→
⏱STT 取得逐字時間軸
→
✨播放 + Karaoke highlight
語音選擇

6 種 Aura 語音（男/女、不同語氣風格）· 語速 0.75x–1.25x 可調

播放控制

進度條 seek · 暫停繼續 · 分頁標題顯示進度 · STT 失敗自動 fallback
